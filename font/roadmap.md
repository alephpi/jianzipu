# 简式（simple form）
## 徽位指法
- 指法与数字的相对位置（用连字解决）
  - 大
  - 食
  - 中
  - 名
  - 跪 
- 数字与数字的相对位置（用连字解决）
- 徽位指法为空、无数字（散音与就）的情况
  - 散
  - 就
## 弦序指法
- 指法与数字的相对位置（用连字解决）
  - 基础右手八法
    - 托
    - 劈
    - 勾
    - 剔
    - 抹
    - 挑
    - 打
    - 摘
    - 托劈（连字）
    - 勾剔（连字）
    - 抹挑（连字）
    - 打摘（连字）
  - 其他指法
    - 历
    - 蠲
    - 轮
    - 半轮
    - 琐
    - 长琐
    - 如一
    - 伏
    - 如一声
    - 滚
    - 拂
    - 至
    - 滚拂
    - 全扶
    - 拨
    - 剌
    - 剌伏
    - 打圆
    - 搯起
    - 抓起
    - 带起
    - 掩
    - 虚掩
    - 推出
    - 不动
  - 带绰注的情况（左右相对位置）+ 指法与数字连字

## 布局
0. 单数字
1. 弦位指法 + 数（1/2）
2. 徽位指法 + 数（1/2）
3. 徽位指法 + 数（1/2）+ 弦位指法 + 数（1/2）前三者并入 3
4. 徽位指法 + 数（1/2）+ 注/绰 + 弦位指法 + 数（1/2）
